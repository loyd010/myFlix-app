Documentation Page

This API will be used to retrieve information about movies, including movie details, directors, genres, etc., from a database when queried. The API will be accessed via HTTP methods like GET and POST.

The table below shows the various requests that may be made to this app, as well as other details about the requests.

Description Endpoint URL HTTP Method Query Paramaters Request Body Data Format Response Body Data Format
Return a list of all movies to the user /movies GET None None A JSON object holding data about all of the movies, ex. { Title: "", Description: "", Genre: {Name: "", Description: ""}, Director: {Name: "", Bio: ""}}
Return data about a single movie by title /movies/:title GET Movie title None A JSON object holding data about a single movie, ex. { Title: "", Description: "", Genre: {Name: "", Description: ""}, Director: {Name: "", Bio: ""}}
Return data about a genre by name /movies/genre/:genreName GET Name of the genre None JSON object holding data about a single genre, ex. { Name: "", Description: "" }
Return data about a director by name /movies/director/:directorName GET Name of the director None JSON object holding data about a single director, ex. { Name: "", Bio: "", Birth year: "", Death year: "" }
Allow new users to register /users POST none JSON object holding data about the user to add, ex. {username: "Tara1234", password: "1234", email: "loyd@gmail.com", Birthday: "", FavoriteMovies: [] } JSON object holding data about the user that was added, including an ID: {id: 1, username: "Tara1234", password: "1234", email: "loyd@gmail.com", Birthday: "", FavoriteMovies: [] }
Allow users to update their user information (username) /users/:Username PUT Username and credentials JSON object with data to be updated, ex. {username: "23ryan", password: "9876%$#"} JSON object showing updated user data, ex. {id: 2, username: "23ryan", password: "9876%$#", email: "ry23@gmail.com", Birthday: "", FavoriteMovies: [] }
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST Movie ID None A text message indicating that the movie was successfully added
Allow users to remove movies from their list of favorites /users/:Username/movies/:MovieID DELETE Movie ID None A text message indicating that the movie was successfully removed
Allow existing users to deregister /users/:Usernames DELETE Username and credentials None A text message indicating whether the user successfully deregistered
